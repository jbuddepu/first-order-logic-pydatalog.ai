# First Order Logic using PyDatalog

Demonstrates first-order logic and recursive predicates using PyDatalog.

## Example
- Defines parent relationships
- Computes ancestor relationships using transitive closure
- Queries for all ancestors of a given person

## Usage
```bash
python first_order_logic_pydatalog.py
```

## Output
Lists all ancestors of 'charlie'.

## Requirements
```
pyDatalog
```

## Concepts
- **Facts**: Direct relationships (parent relationships)
- **Rules**: Derived relationships (ancestor relationships)
- **Recursion**: Transitive closure for multi-generation ancestry
